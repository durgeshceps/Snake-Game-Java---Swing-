# 🐍 Snake Game (Java - Swing)

A simple **Snake Game** built using **Java** and **Swing GUI**.

---

## 🎮 Features
- Classic snake gameplay
- Apples increase length & speed
- Score counter
- Game Over screen
- Press **SPACE** to restart
- Smooth movement and simple graphics
